# Installation

Install Foundation with Composer.

```shell
composer require raxos/foundation
```

## Requirements

- PHP 8.5 or newer.
- The following PHP extensions:
    - `ext-dom`
    - `ext-intl`
    - `ext-json`
    - `ext-mbstring`
    - `ext-openssl`
    - `ext-simplexml`

## Raxos dependencies

Foundation is the base package of the Raxos ecosystem. It requires no other Raxos packages, only the PHP extensions above and the `jetbrains/phpstorm-attributes` package for editor hints.

Return to the [Foundation introduction](/foundation/).
